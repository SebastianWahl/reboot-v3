Profil cognitif — 3 approches

Quelle structure te parle le plus ?

A

Carte des registres

Le profil est organisé autour des 4 registres cognitifs. Chaque test futur vient enrichir un ou plusieurs registres. La "progression" se lit comme un taux de complétude — combien de registres sont explorés en profondeur.

Reptilien ●●●○ Instinctif ●●○○ Émotionnel ●○○○ Rationnel ●●●●

+ synthèse croisée Doctor Claude au centre

B

Document vivant

Le profil est un document permanent avec des sections fixes (Qui je suis, Mes forces, Mes priorités, Mes pratiques). Chaque test complète et enrichit ces sections. Le radar et la synthèse sont toujours visibles en haut.

📊 Snapshot — Radar + score global
🧠 Qui je suis — Synthèse consolidée
💪 Mes forces — Issues de tous les tests
🎯 Mes priorités — Actions en cours
C

Vue 2 colonnes : profil + complétude

Colonne gauche = profil actuel (radar, synthèse, priorités). Colonne droite = tableau de bord de complétude — les tests disponibles, ceux faits, ceux à faire. Une barre de progression "Profil complété à X%".
